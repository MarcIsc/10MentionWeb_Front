rowspan colspan

rowspan est un attribut qui permet d'indiquer le nombre de ligne qu'une cellule <td> de tableau <table> peut couvrir. Cet attribute est utile si vouys souhaitez qu'une cellule s'étende verticalement sur plusieur ligne. Exemple, si on met rowspan="5" alors notre cellule occupera l'espace de 5 cellule verticalement. Cependant, il est important de noter que lorsqu'on utilise cette manipujlation, il faut retirer les td correspondant au autres lignes.
